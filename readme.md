
## 关于Laravel Admin CMS

基于Laravel的后台内容管理开发系统，后台基于layui-mini前端主题。能够帮助你节约开发基础系统功能，及一些常规功能插件。是一个快速上手，项目基础开发，外包开发首选CMS。

- 后台系统常用功能

- 简单上手，快速开发

- 更多插件功能，一键安装扩展

- 完全开源，更好的进行业务开发

- 定期更新laravel主版本

希望本项目可以为你，节约开发时间。
